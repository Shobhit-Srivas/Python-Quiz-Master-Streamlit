# 🐍 Python Quiz Master

An interactive Python Quiz Application built using Streamlit.

This project helps beginners test their Python fundamentals through 10 carefully selected questions and provides instant feedback, score calculation, and performance analysis.

---

## 🚀 Features

✅ Interactive Quiz Interface

✅ 10 Python Beginner Questions

✅ Instant Score Calculation

✅ Percentage Calculation

✅ Performance Evaluation

✅ User-Friendly UI

✅ Real-Time Feedback

✅ Streamlit Powered Web Application

---

## 📸 Application Preview

The application asks users Python-related questions and evaluates answers instantly.

Example:

1. Who developed Python?
2. What keyword is used to define a function?
3. What symbol is used for comments?

and many more...

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Shobhit-Srivas/Python-Quiz-Master-Streamlit.git
```

Move into project folder:

```bash
cd python-quiz-master-streamlit
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
